# Estudo Dirigido 04

Projeto desenvolvido na disciplina TE&SE utilizando:

- LaunchPad F28379D
- Comunicação SCI
- DAC e ADC
- Python para geração e aquisição de sinais

## Objetivos

- Enviar sinais do Python para o C2000
- Reconstruir sinal no DAC
- Adquirir via ADC
- Retornar dados ao Python
- Realizar FFT e análise espectral
